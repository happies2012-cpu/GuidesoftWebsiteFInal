## 📞 Support & Troubleshooting

### Common Issues

**Can't access app?**
```bash
npm run start:all
```

**Login not working?**
```bash
# Check backend: http://localhost:3001/api/health
```

**Database issues?**
```bash
npx prisma migrate reset
node prisma/seed.js
```

**Port conflicts?**
```bash
# Kill processes
lsof -ti:8080 | xargs kill -9
lsof -ti:3001 | xargs kill -9
```
